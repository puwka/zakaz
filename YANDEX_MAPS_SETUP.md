# Настройка Яндекс.Карт

## Инструкция по настройке

### 1. Получение координат адреса

Чтобы получить координаты вашего адреса:

1. Откройте [Яндекс.Карты](https://yandex.ru/maps)
2. Найдите ваш адрес
3. Кликните правой кнопкой мыши на нужное место
4. Выберите "Что здесь?"
5. Скопируйте координаты (формат: долгота, широта)

Или используйте [Геокодер Яндекс.Карт](https://yandex.ru/dev/geocoder/doc/ru/)

### 2. Обновление координат в коде

Откройте файл `app/(public)/contacts/page.tsx` и найдите строку:

```tsx
<YandexMap 
  address={companyInfo.address}
  center={[37.6173, 55.7558]} // Замените на ваши координаты
  zoom={15}
/>
```

Замените `[37.6173, 55.7558]` на ваши реальные координаты.

**Формат:** `[долгота, широта]`

Пример для Москвы:
- Долгота: 37.6173
- Широта: 55.7558

### 3. Опционально: API ключ

Для расширенной функциональности можно получить API ключ:

1. Перейдите на [Яндекс.Кабинет разработчика](https://developer.tech.yandex.ru/)
2. Создайте новый ключ для JavaScript API
3. Обновите URL в `components/YandexMap.tsx`:

```tsx
script.src = 'https://api-maps.yandex.ru/2.1/?apikey=ВАШ_КЛЮЧ&lang=ru_RU';
```

**Примечание:** Без API ключа карта также работает, но с некоторыми ограничениями.

### 4. Тестирование

После настройки координат:
1. Обновите страницу контактов
2. Проверьте, что карта загружается
3. Убедитесь, что метка находится в правильном месте
4. Проверьте работу ссылки "Открыть в Яндекс.Картах"
