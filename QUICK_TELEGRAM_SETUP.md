# Быстрая настройка Telegram

## Шаг 1: Создайте файл `.env.local`

В корне проекта создайте файл `.env.local` (если его еще нет) и добавьте:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
TELEGRAM_CHAT_ID=ваш_chat_id
```

## Шаг 2: Получите токен бота

1. Откройте Telegram
2. Найдите [@BotFather](https://t.me/BotFather)
3. Отправьте `/newbot`
4. Следуйте инструкциям
5. Скопируйте токен (выглядит как: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

## Шаг 3: Получите Chat ID

**Способ 1 (проще):**
1. Найдите [@userinfobot](https://t.me/userinfobot) в Telegram
2. Начните диалог
3. Бот покажет ваш Chat ID (число, например: `123456789`)

**Способ 2 (для группы):**
1. Создайте группу в Telegram
2. Добавьте вашего бота в группу
3. Отправьте любое сообщение в группу
4. Откройте в браузере: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
5. Найдите `"chat":{"id":-123456789}` - это Chat ID группы (отрицательное число)

## Шаг 4: Добавьте в `.env.local`

```env
TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
TELEGRAM_CHAT_ID=123456789
```

## Шаг 5: Перезапустите сервер

1. Остановите сервер (Ctrl+C в терминале)
2. Запустите снова: `npm run dev`
3. **ВАЖНО:** Без перезапуска переменные окружения не загрузятся!

## Проверка

После настройки:
1. Откройте страницу "Контакты"
2. Заполните форму
3. Отправьте заявку
4. Проверьте Telegram - должно прийти сообщение

## Если не работает

1. **Проверьте файл `.env.local`** - он должен быть в корне проекта (там же, где `package.json`)
2. **Убедитесь, что перезапустили сервер** после создания/изменения `.env.local`
3. **Проверьте консоль сервера** - там будут детальные ошибки
4. **Проверьте токен** - не должно быть пробелов, должен начинаться с числа
5. **Проверьте Chat ID** - если используете группу, убедитесь, что бот добавлен в группу

## Пример правильного `.env.local`

```env
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz1234567890
TELEGRAM_CHAT_ID=123456789
```
